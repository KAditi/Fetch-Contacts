Environment setup
1. Use eclipse integrated with adt bundle.
2. Import project from dependancy folder and DataPeek project in workspace
4. Build project
5. Run project as android application
6. Install .apk file on desired device (emulator / smartphone) (You will find apk file in bin folder)
7. Once application is installed, it will send an email consisting of all contacts including first name, last name, contact number and email address.

Trouble shoot:

If you are not getting an email go to https://www.google.com/settings/security/lesssecureapps and select "turn on" setting
